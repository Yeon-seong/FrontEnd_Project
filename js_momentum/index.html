CSS와 javascript 파일을 HTML 파일에 불러오기

css, javascript 파일을 브라우저에서 열면
해당 파일에서 작성한 내용(코드)만 화면에 출력된다.
css, javascript는 파일을 연다고 실행되지는 않는다.

HTML을 통해서 css, javascript 파일을 브라우저에서 실행할 수 있다.
브라우저는 HTML을 열고, HTML은 CSS와 javascript를 가져온다.
css, js 파일은 브라우저에서 여는 게 아니라
index.html HTML 파일에 불러온 후 열어야 실행된다.

css, javascript 파일을 불러오는 위치

css 파일은 맨 위의 head 부분에서 가져오며,
자바스크립트 파일은 주로 끝에서 가져온다.
브라우저는 실행되어야 하는 파일이 있다는 것이 확인되면
브라우저는 HTML을 실행해서 해당 파일을 불러온다.
